🔧 Technical Information

📊 Demo Data Features

  • • 5 Sierra Leone councils with real data
  • • Sample projects across different categories
  • • 4 user roles with different permissions
  • • Local browser storage (no database required)
  • • SLL to USD currency conversion
  • • Real-time dashboard updates

🛠️ Technology Stack

  • • Frontend: React 18 + TailwindCSS
  • • Authentication: Demo mode + Supabase ready
  • • Server: Python HTTP server (port 3000)
  • • Storage: Browser localStorage
  • • Icons: Emoji + SVG icons
  • • Responsive design for all devices

📖 Usage Instructions

  1. 1
    Start the demo: Click "Go to Login Page" above or navigate to http://localhost:3000/login.html
  2. 2
    Choose a role: Use any of the demo accounts listed above to explore different user experiences
  3. 3
    Explore features: Each role has access to different modules and capabilities
  4. 4
    Create test data: Try creating new users, councils, and projects to see the system in action
